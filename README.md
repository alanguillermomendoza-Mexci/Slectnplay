Slectnplay is a Windows desktop Text-to-Speech application built for reading-heavy workflows.

It converts selected text into audio instantly when the user releases the mouse, helping reduce visual fatigue and making it easier to move through dense information without staying locked to the screen.

## What It Is For

Slectnplay is designed for people who spend long periods reading, reviewing, or comparing text on a computer.

It is useful for:

- Code and technical snippets
- Logs and diagnostic output
- Documentation and reference material
- Technical articles and specifications
- Long-form reading sessions
- Repetitive review tasks where listening is faster than rereading

## Key Capabilities

- Instant playback from selected text
- Desktop-first workflow for Windows
- Offline Text-to-Speech using local voice assets
- Bilingual reading support for English and Spanish workflows
- Technical reading support for symbols, operators, and common programming tokens
- No cloud processing required for text-to-speech playback

## Privacy

Slectnplay is designed to run locally.

The application does not collect, transmit, store, sell, or share user data. Text selected by the user is processed on the device for the purpose of generating audio output.

See `docs/privacy.md` for the public privacy statement.
