# Store Listing Alignment

## Core Product Message

Slectnplay is a Windows desktop Text-to-Speech application that converts selected text into audio instantly when the user releases the mouse. It is designed to reduce visual fatigue and support reading-heavy workflows.

## Key Claims

- Instant playback from selected text.
- Windows desktop workflow.
- Useful for documentation, logs, code-like content, technical text, and long-form reading.
- Offline processing using local voice assets.
- Designed to help users listen through dense content instead of relying only on visual reading.

## Privacy Position

Slectnplay does not collect, store, transmit, sell, or share user data. Text processing and audio generation are performed locally on the user's device, without cloud processing or analytics.

## Repository Position

This GitHub repository is for public portfolio and product documentation only. The Slectnplay application is closed-source, and this repository does not include proprietary source code, local voice models, build outputs, installers, signing material, or internal implementation details.
