# Privacy Policy

Slectnplay is a desktop application that converts selected text into speech locally on the user's device.

The application does not collect, store, transmit, or share any personal information.

All text processing and audio generation is performed offline using local models. No data is sent to external servers.

The application only accesses text explicitly selected by the user for the purpose of generating audio output.

Slectnplay does not track user behavior, does not use analytics, and does not include advertising or third-party data collection.
