# Slectnplay Initial Release

Slectnplay is a Windows desktop Text-to-Speech application that converts selected text into audio instantly on mouse release.

This release represents the initial public product version prepared for Microsoft Store distribution.

The application is fully offline, does not collect user data, and is designed to improve reading-heavy workflows by reducing visual fatigue.

Note: This repository does not include binaries or source code. It is for product documentation and portfolio purposes only.

# Lanzamiento inicial de Slectnplay

Slectnplay es una aplicación de escritorio para Windows que convierte texto seleccionado en audio de forma instantánea al soltar el mouse.

Este lanzamiento representa la versión pública inicial del producto preparada para distribución en Microsoft Store.

La aplicación funciona completamente offline, no recolecta datos del usuario y está diseñada para mejorar flujos de trabajo con alta carga de lectura reduciendo la fatiga visual.

Nota: Este repositorio no incluye binarios ni código fuente. Es únicamente para documentación de producto y portafolio.
